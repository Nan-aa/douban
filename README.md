
# 技术栈
vue2 + vuex + vue-router +  ES6+ fetch + sass + flex + svg+ vantUI

**版本:**

vue: ^2.6.11,

 vue-router: ^3.1.6,

 vuex: ^3.1.3"，

vant: 1

node:12.16.2.0

# 项目运行 

yarn serve (或者 npm run serve)
